Expense-Tracker CLI
A simple, zero-dependency command-line expense tracker written in Python 3.
Keep tabs on your daily spending by adding, listing and filtering expenses.
Features

    Add an expense with amount and category
    List all recorded expenses
    Show the running total
    Filter expenses by category
    Exit anytime

All expenses are kept in a list in memory while the program is running; no persistence is provided.

## How to run

```bash
python main.py
```
